cleanmarkup Cookbook
====================
Install the clean markup module on a D7 website.

Requirements
------------

Usage
-----
#### cleanmarkup::default
Just include `cleanmarkup` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[cleanmarkup]"
  ]
}
```
